## <mark style="color:green;">Evil-WinRM</mark>

Tool to connect remotele to a Windows WinRM instance

* Install

{% code overflow="wrap" lineNumbers="true" %}
```bash
sudo apt install evil-winrm
```
{% endcode %}

***

* Acces  to WinRM

{% code overflow="wrap" lineNumbers="true" %}
```bash
sudo evil-winrm -i $ip -u $user -p $password
```
